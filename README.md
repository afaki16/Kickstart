🚀 Kickstart Template
Bu template ile hızlıca .NET Web API + Nuxt.js projesi oluşturabilirsiniz.
📁 Proje Yapısı
Kickstart/
├── backend/                 # .NET Web API
│   ├── {{PROJECT_NAME}}.API/
│   ├── {{PROJECT_NAME}}.Application/
│   ├── {{PROJECT_NAME}}.Domain/
│   ├── {{PROJECT_NAME}}.Infrastructure/
│   └── {{PROJECT_NAME}}.sln
├── frontend/               # Nuxt.js Frontend
│   ├── package.json
│   ├── nuxt.config.ts
│   └── ...
└── setup.bat              # Kurulum scripti
